软件安装

  • nodejs （如果是从上面的下载地址下载的是一个压缩包，记得解压）

    • 打开cmd验证是否安装完成

      输入一下两条指令

      node -v
npm -v

      出现下图结果说明安装成功

  • git

​ 打开cmd输入下面命令验证是否安装成功

git --version

​ 出现一下结果表示安装成功

安装 hexo

  • 新建一个文件夹
  • 进入该文件夹，右击选择 Git Bash Here

出现如下命令窗口，以下的命令均在此窗口完成

  • 输入以下命令 

    npm i hexo-cli -g

    • 如果没有科学上网的手段，npm使用起来会很慢

      可以通过给 npm 配置国内镜像源解决，方法如下

      • 输入以下命令

        npm install -g cnpm --registry=https://registry.npm.taobao.org

        出现以下结果表示配置成功

        此后均使用 cnpm 代替 npm 指令，下文不在做说明

    • 重新输入上述命令，使用 cnpm 替换 npm

      cnpm i hexo-cli -g
      • 出现以下结果表明成功

      • 输入以下命令再次验证

        hexo -v

        出现以下结果表明安装成功

  • 输入以下命令初始化文件夹

    hexo init
    • 出现以下结果

    • 文件夹里面会出现以下内容

  • 输入以下命令安装必备组件

    cnpm install

    • 出现以下结果

  • 至此，本地的网站配置全部完成，输入以下命令打开本地服务验证

    hexo g && hexo s

    • 在浏览器打开下图红圈中的链接

    • 浏览器会出现此页面

将网站部署至Coding代码托管平台

  • 注册并登录 Coding 平台

    Coding官网地址

    注册过程省略

  • 新建项目

    • 点击项目设置

    • 再点击功能开关

    • 打开下路红圈中的 持续部署 功能

    • 退出项目设置

    • 选择静态网站

    • 第一次需要进行实名认证

    • 点击立即发布静态网站

    • 红框内的网址即我们的博客地址

    • 现在是404页面，一会连接我们的本地仓库就好

连接本地与Coding

配置Git

  • 输入以下命令，将其中引号处日光自己的用户名及邮箱

    git config --global user.name "KG-Lz"
git config --global user.email "1543404489@qq.com"

  • 输入以下代码生成SSH密钥

    ssh-keygen -t rsa -C "1543404489@qq.com"
    • 出现下图敲回车

    • 继续敲回车

    • 再来一次

    • 出现以下内容则完成

    • 输入以下代码将SSH密钥复制出来备用

      cat ~/.ssh/id_rsa.pub

      • 红框即我们的密钥复制备用

Coding填写密钥

  • 右上角进入个人设置

  • 选择 SSH公钥

  • 新增公钥

    • 将上面复制备用的公钥内容填写在此处，勾选永久有效

  • 验证SSH连接

    • 输入以下代码
    ssh -T git@e.coding.net

仓库部署

  • 输入以下代码安装部署需要的工具

    cnpm i hexo-deployer-git
