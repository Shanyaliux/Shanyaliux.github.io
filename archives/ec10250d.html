安装Manjaro之后要做的一些事
发表于 2020-04-07| 更新于 2020-04-12|Manjaro
字数总计:375|阅读时长: 1 分钟
|阅读量:
为什么是Manjaro？
对我来说只有一点，就是 ***aur*** 。
本文记录的是我安装完Manjaro之后做的一些事。
ps： Manjaro根据不同的桌面环境有三个版本，我的是gnome版本

更换源

终端输入一下两个命令

sudo pacman-mirrors -i -c China -m rank
sudo pacman -Syy

这会显示几个速度快的源，选最快的那个。
当然也可以自己添加源到配置文件。
在/etc/pacman.conf的最后一行添加：

[archlinuxcn]
SigLevel = Optional TrustedOnly
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch

然后终端运行命令：

sudo pacman -Syy
sudo pacman -S archlinuxcn-keyring

常用软件

yay

它可以代替pacman安装软件

sudo pacman -S yay base-devel

搜狗输入法

按照我的操作可以实现大多数地方的中文输入，有些软件不能输入：网易云（ps：主要这个我不是那么的在乎）

sudo pacman -S fcitx-im #全部安装
sudo pacman -S fcitx-configtool
sudo pacman -S fcitx-sogoupinyin
sudo pacman -Rsn fcitx-im fcitx-configtool
sudo pacman -S fcitx-lilydjwg-git fcitx-sogoupinyin
sudo pacman -S fcitx-configtool fcitx-qt5
sudo pacman -S fcitx5

在~/.xprofile添加用户环境

export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS="@im=fcitx"

谷歌浏览器

sudo pacman -S google-chrome

git

sudo pacman -S git

vscode

sudo pacman -S visual-studio-code-bin

网易云音乐

yay -S netease-cloud-music

wps

yay -S wps-office
yay -S ttf-wps-fonts
